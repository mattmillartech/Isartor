# Isartor Governance

## Independence Commitment

Isartor is maintained by an independent developer.
There is no parent company, no VC funding, and no
commercial platform that Isartor feeds into.

## License Stability

Isartor is licensed under Apache 2.0. This license will
not be changed to a proprietary or source-available model.
This is a permanent commitment.

This commitment exists because the infrastructure OSS
community has been burned by relicensing events
(HashiCorp/Terraform, Redis, Elasticsearch). Isartor
explicitly rejects this path.

## Decision Making

All architecture decisions are made by the maintainer
with input from the community via GitHub Issues and
Discussions. There is no steering committee or board.

## Sustainability

The enterprise license tier (compliance features) is how
this project sustains itself. Free OSS core. Paid
enterprise features. No VC pressure. No acquisition path.
